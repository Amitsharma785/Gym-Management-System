## Project Overview

The Gym Management System is a SQL-based database project designed to manage gym operations efficiently.

The system stores and manages:
- Member information
- Trainer details
- Workout plans
- Payment records
- Attendance records

This project demonstrates relational database design and SQL query operations using MySQL.

## Technologies Used

- MySQL
- MySQL Workbench
- SQL
- dbdiagram.io
- GitHub

## Database Tables

### Members
Stores member details like name, age, phone number, and joining date.

### Trainers
Stores trainer information and specialization.

### Workouts
Stores workout plans assigned to members by trainers.

### Payments
Stores payment records and transaction dates.

### Attendance
Stores member attendance status.
